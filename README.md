\# Doc.RAG



A full-stack Retrieval-Augmented Generation (RAG) application for chatting with PDF documents using semantic search, reranking, and grounded LLM responses.



\## Features



\- PDF upload and parsing

\- Text chunking and embedding generation

\- FAISS-based vector retrieval

\- Reranking for better relevance

\- Context-aware chat with conversation history

\- Streaming responses

\- Document deletion from vector store and metadata

\- Modern UI built with Next.js and shadcn/ui



\## Tech Stack



\### Frontend

\- Next.js

\- TypeScript

\- Tailwind CSS

\- shadcn/ui



\### Backend

\- FastAPI

\- FAISS

\- SQLite

\- PyPDF

\- OCR fallback support

\- Ollama / Gemma
